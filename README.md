# 学生体测成绩管理系统 (Sport Score Management System)

一个基于 C 语言开发的高效、简洁的学生体能测试成绩管理系统。本项目采用**双向链表**（或单链表，视具体实现而定）数据结构，支持对学生体测数据的增删改查、排序、导入导出等核心功能。

## 🚀 主要功能

- **学生信息管理**: 支持学生姓名、学号、班级、年级、性别、身高、体重等基础信息的维护。
- **体测项目追踪**: 全面支持 7 项国家宪法规定的体测项目评分：
  - BMI (身体质量指数)
  - 肺活量
  - 坐位体前屈
  - 50米跑
  - 立定跳远
  - 引体向上 / 仰卧起坐
  - 1000米 (男) / 800米 (女)
- **批量数据导入**: 能够解析特定格式的 `.txt` 文件，快速将大量学生信息录入系统。
- **智能排序系统**: 
  - 按总成绩排序。
  - 按特定单项成绩排序。
  - 按学生体重排序。
- **数据导出功能**: 支持将系统内的学生体测数据一键导出为 `.csv` 格式，方便使用 Excel 等工具二次处理。
- **持久化存储**: 自动读取及保存数据到本地文件，确保数据不丢失。

## 📂 文件结构

- `mian.c`: **主程序入口**，负责 UI 交互逻辑、菜单展示及功能调用。
- `information.h`: **核心头文件**，包含了 `struct student` 结构体定义以及所有链表操作和业务处理函数的实现。
- `students.txt`: **数据存储文件**，系统默认加载及保存的文本数据。
- `current_func.txt`: **功能备份/开发记录**，记录了关键函数（如 `import_from_custom_txt`）的最新实现。

## 🛠 开发环境与编译

### 环境要求
- **操作系统**: Windows / macOS / Linux
- **编码建议**: 本项目源码采用 **GBK** 编码（主要为了兼容旧版 Windows 终端）。在 Visual Studio Code 等编辑器中，请确保以 `simplified chinese (gbk)` 方式打开，避免中文乱码。
- **编译器**: GCC 或任何标准 C 编译器

### 编译与执行

在项目根目录下，使用以下命令进行编译：

```bash
gcc mian.c -o sport_management
```

运行程序：

```bash
./sport_management
```

## 📖 使用说明

1. **首页菜单**: 启动程序后，根据数字提示选择对应功能。
2. **导入数据**: 若已有 `students.txt`，可选择“导入”功能快速填充系统。
3. **录入与修改**: 按照终端提示输入具体数值。
4. **排序显示**: 可自由选择按总分或单项分进行升序/降序排列。
5. **保存退出**: 退出程序前，系统会自动提示是否保存当前修改。
